# 📈 Mebar Professional Trader (v7.1)

Mebar Trader is a real-time, high-performance desktop financial terminal developed to track personal portfolios, analyze live market data, and generate AI-driven sentiment analysis on financial news.

![Mebar Trader Interface](BURAYA_EKRAN_GORUNTUSUNUN_LINKINI_KOYACAGIZ)

## 🚀 Key Features
* **Real-Time Market Tracking:** Fetches live data for stocks, funds, and currencies.
* **AI Sentiment Analysis:** Uses a custom Python NLP backend to read live financial news and score them from 0-100 (Positive/Negative).
* **High-Performance UI:** Built with C++ and ImGui for a seamless, async, crash-free desktop experience.
* **Risk Management:** Interactive Target Profit and Stop-Loss charts.
* **Excel Export:** One-click CSV reporting for portfolio management.

## 🛠️ Technologies Used
* **Frontend:** C++, ImGui, ImPlot, GLFW, OpenGL3
* **Backend:** Python, Flask, YFinance, BeautifulSoup4 (Web Scraping)
* **Architecture:** Multi-threading, Mutex locking, RESTful API communication.

## ⚙️ How to Run
1. Download or clone this repository.
2. Simply double-click the **`Baslat.bat`** file.
3. The batch file will automatically spin up the Python backend server in the background and launch the C++ UI smoothly!
